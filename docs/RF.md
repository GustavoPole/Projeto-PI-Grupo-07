# ⚙️ Requisitos Funcionais (RF)

| ID | Requisito | Descrição |
|:--- |:--- |:--- |
| **RF01** | **Gestão de Perfil** | O sistema deve coletar idade, peso, altura, sexo e nível de atividade física. |
| **RF02** | **Geração de Dieta** | O sistema deve gerar uma dieta semanal baseada nos dados do RF01 via IA. |
| **RF03** | **Filtros Alimentares** | O usuário deve marcar restrições (ex: Vegano, Low Carb, Intolerante a Lactose). |
| **RF04** | **Log de Refeições** | O usuário deve poder dar "check" em cada refeição sugerida do dia. |
| **RF05** | **Controle de Água** | O sistema deve permitir o registro de copos de água e calcular a meta diária. |
| **RF06** | **Evolução de Peso** | O sistema deve gerar um gráfico com base no histórico de peso inserido pelo usuário. |
