Título: [RF-001] 
Descrição:

O sistema deve permitir que o usuário realize o login na aplicação utilizando e-mail e senha ou provedores sociais (Google/Apple).

Critérios de Aceite:

[ ] O usuário deve conseguir criar uma conta nova.

[ ] O usuário deve conseguir recuperar a senha via e-mail.

[ ] Após o login, o usuário deve ser redirecionado para a HomeScreen.

[ ] Validar se o e-mail inserido possui formato válido.
