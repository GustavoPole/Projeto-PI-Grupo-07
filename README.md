# Projeto-PI

## Descrição do Projeto
O projeto se chama DietHub e se trata de um aplicativo mobile que tem a funcionalidade de criar dietas controladas para os usuários com o auxilio de uma Inteligencia Artificial integrada dentro do App que fornerá apoio aos usuários montando uma dieta equilibrada e controlada. Caso o usuário não goste da dieta montada ele pode acionar a IA que realizará a troca da dieta ou dos alimentos por outros equivalentes para não afetar a alimentação do usuário.

## Publico
O aplicativo é direcionado a todos os publicos que querem começar um rotina mais fitness sem ter o gasto de um nutricionista. (Recomendamos um Nutricionista caso a pessoa queira um acompanhamento mais completo)
